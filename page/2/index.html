wowza-streaming-engine串流輸出wowza-streaming-cloud-CDN

發表於 2018-11-23 更新於 2020-04-01

本文章將介紹如何使用 wowza engine 串流輸出到 wowza cloud CDN

1. 首先登入 wowza cloud 來到 Advanced 頁面

2. 再次點擊 Advanced 選擇 Stream Targets

3. 點擊 Add Target 選擇 Wowza CDN - HLS 新增 CDN

4. 設定名稱其他選項預設，點選 Add 加入

5. 下圖是設定好畫面紅框為連接代碼，帶有1的代碼用於將目標連接到 Wowza Streaming Engine

6. 接下來到 wowza streaming engine → Transcoder → 點擊 Enable Transcoder 開啟轉碼

7. 開啟後點選 Restart Now 重新整理

8. 重新整理完後，記得要回到 Stream Files 重新串流

wowza 串流文章傳送們

9. 看到下圖畫面就是設定成功了 ( 目前是使用預設值，轉碼設定會再做一篇文章 )

10. 再來到 wowza streaming engine → Stream Targets → Add Stream Targers 新增輸出

11. 選擇 wowza streaming cloud 把串流輸出到剛剛 wowza 雲端建好的 cdn

12. 輸入名稱 Stream Target Name，輸入連線代碼 Connection Code ( 連線代碼為在 wowza cloud 建立好 cdn 給的代碼 )

12. 輸入好代碼後點擊 Check Code 輸入建好的串流名稱，點選 Add this target

13. 看到下圖畫面就是成功轉碼並輸出到 wowza cloud cdn 了

14. 接下來回到 wowza cloud 複製 Apple HLS Playback URL

15. 開啟撥放器網路串流，把剛剛複製的 Apple HLS Playback URL 貼上，然後撥放

16. 如下圖出現畫面就是成功了

wowza-stream-engine串流輸出到wowza-streaming-cloud

發表於 2018-11-23 更新於 2020-04-01

本文章將介紹 wowza-engine 串流輸出到 wowza-cloud 的基本操作

首先登入 wowza streaming cloud

點選 Live Streams

點擊左上方或下方的 Add Live Stream 新增直播

輸入 live stream 選擇地區

選擇輸入進來的方式，本文章使用 wowza streaming engine，下方設定分別為長寬、是否錄製成影片、隱藏字幕類型


撥放設置依序份別為：撥放瀏覽器選擇、撥放器寬度、直播開始前撥放器顯示時間倒數、撥放前畫面顯示圖片、撥放中顯示圖片、圖片放置位置

託管設置依序分別為：是否讓 wowza 託管撥放是頻網頁、標題、託管葉面上放置圖片、說明

最後一頁會把你的設置列出來給你看

全部設定好會進入下圖畫面，複製紅框代碼

接下來到 wowza streaming engine → Stream Targets 選擇 wowza streaming cloud

輸入 Stream Target Name、再輸入從 wowza streaming cloud 拿到的代碼，輸入完點擊 Check Code 做確認

點擊完 Check Code 會出現 Source Stream Name 輸入你 Incoming Stream 串流的名稱

再來回到 wowza streaming cloud → live streams 點選下圖紅框的 Start Live Stream 開始直播

點擊 Start Live Stream 會出現下圖提示點擊 Start

接下來會跑讀條成功就出現 Ready


下圖出現畫面直播就開始了，下方紅框為手機使用串流網址

也可以點選分頁的 Playback 觀看

或是點選 Monitior 下圖點擊紅框圖案會另外開啟一個分頁直播畫面


PS.如果有停止或是重啟動作，會需要再重新輸入一次代碼

要到 Video Source and Transcoder 下方 Regenerate Connection Code 重新取得代碼


到這邊就完成了

wowza串流輸出Youtube直播

發表於 2018-11-15 更新於 2020-04-01

本文章將介紹使用 GoCoder 串流到 wowza 在串流輸出到 youtube 直播

ps.youtube 輸出方法其他串流方式也是相同，本文章使用比較方便的手機 GoCoder 介紹

首先開啟 youtube 點擊右上角進行直播

下圖為直播設定畫面


接下來到 wowza → Stream Targets → Add Stream Targts 設定輸出

選擇 youtube

設定參數 Stream Targets Name：隨意命名、Source Stream Name：Stream File Name

Destination Application Name 對照 youtube 的直播設定畫面 → 編碼器設定 → 伺服器網址的 live2、Destination Host 對照 youtube 的直播設定畫面 → 編碼器設定 → 伺服器網址的 a.rtmp.youtube.com

到 youtube 直播設定畫面 → 編碼器設定 → 串流名稱/金鑰，點擊顯示會出現金鑰，金鑰重設也是在這邊進行


獲得的金鑰輸入在 Destination Stream Name

設定好後回到 youtube 會發現上方會出現將開始即將開始，等待一下後直播就會正式開始了


畫面有出現就是成功了！

wowza使用GoCoder串流

發表於 2018-11-15 更新於 2020-04-01

本文章將介紹用手機 GoCoder 串流 wowza

首先手機先下載 GoCoder

下圖為開啟畫面

點擊右上做設定 ( 串流 wowza engine 只需要設定 Wowza Stream Engine )

進入 wowza web → Source ( Live )，點擊 wowza GoCoder 裡面有一些參數，除了 IP ( 需要輸入外網 IP ) 其他可以對照輸入


點擊 Wowza Stream Engine 會出現3個需要設定選項

設定 Host：輸入外網 IP，Port：1935 ( wowza 預設串流 port )

設定 Application：wowza live application name，Stream Name：GoCoder 預設都是 myStream ( 更改後 Incoming Streams 會顯示更改後名稱，PS.同樣名稱二支手機會互搶 )

手機 GoCoder Source Authentication 要到 wowza Server → Source Authentication → Add Source 新增 Source


設定 Source Authentication：wowza 所設定的 Source Authentication

點擊右下紅點開始串流

串流後畫面如下圖 ( 一些 GoCoder 其他設定有空會再做文章介紹 )

可以到 Inconing Streams 會發現串流正在在執行 ( 使用 GoCoder 的方式是不會有 Stream Files )

設定完後做測試，打開 VLC 輸入串流網址

下圖有畫面就是成功了

wowza串流輸出original-edge抓取original串流

發表於 2018-11-09 更新於 2020-04-01

如下圖 wowza 串流輸出 original、edge 抓取 original 串流

其實可以直接從 wowza 抓取串流網址，多一台 original 是因為安全，然後再由多台 edge 抓取客端再來抓取，多台 edge 可以分散風險

首先進入 wowza Stream Targets

點擊 Enable Stream Targets 開啟

這時候 wowza 會要求你重新整理點擊右上 Restart Now 重新整理

重新整理過後 Incoming Stream 串流會自己斷掉，要到 Stream Files 重新在串流一次


串流好了後回到 Stream Targets 點擊 Add Stream Targets

選擇一種方式輸出，本文章是使用 RTMP

輸入 Stream Targets Name ( 隨意命名 )、Source Stream Name ( Stream Files Name )、Destination Application Name ( Original 串流位置資料夾名稱 )、Destination Host ( Original IP )、Destination Stream Name ( Original 串流輸出名稱 )

設定好後如下圖，出現 Active 就是成功輸出了

如果設定錯誤會出現 Error

如果出現 Waiting 代表串流沒有輸入可以到 Incoming Streams 看


接下來測試出輸出 original 是否成功，打開撥放器 VLC，輸入串流網址

如下圖出現畫面就成功了

再來是測試 edge 能否抓取 original 的串流，打開撥放器 VLC，輸入串流網址

如下圖出現畫面就成功了

wowza使用ip-camera串流

發表於 2018-11-02 更新於 2020-04-01

本章將講解如何使用 ip camera 串流 wowza

先登入 wowza

下圖為登入後畫面，點選上方的 Application 建立一個新的 live Application

進入 Application 後，點選 Live

輸入名稱

進入設定畫面，下圖紅框部分是允許進入格式，剩下的選項預設 ( 有機會會再做篇文章介紹 )，設定好後點擊 Save 儲存

儲存好後會進入下圖畫面，設定隨時都可以更改 ( 左上 live Application 點選後會進入到一開始進去畫面 )

建立好 live Application 後 ip camera 本文章介紹二個串流輸入的方法

一、 點選左邊 Sources ( Live )，裡面有預設選項讓你選，是依照 ip camera 廠牌做分類，但不是全部廠牌都有沒有要用另一種方法加入

本文章使用 ip camera 牌子是 AXIS

點擊後會需要輸入 stream name 和 camera ip address

加入好會進入下圖畫面 ( Incoming Streams ) 右邊 Bytes In 顯示數字就是畫面有進 wowza 了 ( 沒有可以點擊 Refresh 刷新一下 ) 點擊 Return to Incoming Streams Page 返回上一頁

下圖 Active 代表正在正常運作

如果 ip 或是密碼打錯 port 沒開等等問題，就會出現 Waiting for Stream

到這邊設定就完成了

二、 第二種串流方法點選左邊的 Stream Files 進入後點選上方的 Add Stream Files 新增

輸入 Stream Files name 跟 Stream URI ( URI 如果不知道可以到 ip camera 的官網查 )

加入好會進入下圖畫面 ( Stream Files ) 點擊 Return to Stream Files 回上一頁

這種方式串流只有建立一個新的 Stream Files，Incoming Stream 不會幫你串流起來，點擊下圖紅框圖案串流

點擊後在 MediaCaster Type 選擇你 ip camera 進來的格式，其他選項預設就好

出現如下圖紅框文字就是成功了

再到 Incoming Streams 確認是否輸入

三、 wowza 有測試功能可以看影片是否有串流成功

右上角會有一個 Test Players

點擊後會出現一個播放畫面 server：安裝 wowza 機器 ip ( 如果是本機開啟可以用 127.0.0.1 或內網 IP )、Application：live Application Name、Stream：Stream Name.stream

PS.進入 Incoming Streams 選擇要觀看的 stream 進去後再點 Test Players 數值就會都幫你設定好

也可以使用有串流的播放器做測試 ( 本文章使用的是 VLC )

點擊媒體 → 開啟網路串流

輸入串流網址

如下圖出現畫面就成功了

PS.到 Incoming Streams stream 點進去可以看到 Bytes Out 輸出 ( 只有直接從 wowza 輸出這邊才會顯示 )

到這邊 ip camera 到 wowza 這段就完成了

安裝wowza

發表於 2018-10-29 更新於 2020-04-01

一、什麼是 wowza

Wowza Streaming Engine 是媒體服務器軟件，提供可靠性強，流暢度高的高質量視頻和音頻到任何設備的傳輸。無論是在雲端還是當前操作，Wowza Streaming Engine 都可以提供組件，讓我們不僅可以保證工作流的流暢性還有安全性。

1. Wowza 是一個不論平台、多格式、多屏幕軟件。它接受任何的視頻格式，一次性轉碼，可靠地進行多格式傳輸，並以最大可能的質量傳輸到任何地方的任何連接設備。

2. Wowza 提供豐富的功能專為流暢的現場內容而設計，包括位碼流暢性，DVR 性能和發布功能。

3. 用融合了其他系統和第三方解決方案的外延軟件，自定義你的流線型作品。平衡軟件強大的組件和 APIs，建立支持你持續發展的工作流的解決方案。

4. Wowza Streaming Engine Manager 可以從一個電腦或者任何的移動設備管理，監察和測量視頻和音頻。

二、安裝 wowza for windows

官方下載網址：https://www.wowza.com/pricing/installer

下載下方有安裝需求說明

開始安裝

輸入金鑰

設定 wowza web 登入帳密

設定安裝好開啟 web wowza

設定安裝位置

打開網頁輸入 外網 IP:8088 登入測試 ( 如果是本機 IP 可以輸入 127.0.0.1 或是內網 IP )，出現下圖畫面就是安裝成功了，如果是別台電腦打開 wowza web 防火牆 8088 port 記得開

三、安裝 wowza for ubuntu 16.04

同意完會出現金鑰輸入，輸入金鑰

輸完金鑰，鑰需要輸入帳密 ( 登入 wowza web 用 )，先輸入帳號

輸完帳號會問密碼，輸入2次後 OK

設定安裝完自動啟動 wowza

詢問是否安裝

跑完讀條，出現圖文字就是完成了
