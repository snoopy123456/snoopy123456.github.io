GCP-GoogleCloudPlatform-creat-vm-gui

發表於 2018-06-19 更新於 2020-04-01

本文章將介紹如何使用 GUI 建立 GCP VM

1. 進入 GCP，網址：https://cloud.google.com/

2. 登入 Google 帳號

3. 點擊畫面右上角控制台進入

4. 建立 VM

點選資訊主頁畫面左上方的三條槓導覽選單查看 GCP 產品和服務列表

點選 Compute Engine → VM 執行個體 ( 可點選右邊圖釘把自己常用的工具釘選到最上面 )

點選建立，建立 VM

有建立 VM，點選建立執行個體

(1) 輸入名稱（ 名稱必須為小寫字母、數字或連字號，且結尾須為小寫字母或數字 ）選擇區域與機器類型

PS：如果想固定區域可以到 Compute Engine → 設定，把區域設定好後每次建立 VM 區域預設就會顯示你設定的區域

設定 CPU 跟記憶體，GCP 預設有幫你配好看要高記憶體還是高 CPU，可以直接選擇

也可以點擊自訂 ( 上圖 ) 調整成自己想要的核心數量、記憶體大小

(2) 選擇作業系統跟硬碟類型和容量

點選變更選擇作業系統 image


點選 ▼ 選擇硬碟類型，右邊輸入需要的容量


(3) 設定身分及 API 存取權

(4) 設定管理、磁碟、網路、SSH 金鑰

設定管理頁籤內容

設定磁碟

硬碟部分有做一篇 GCP 硬碟設定介紹，可以到這邊看 文章傳送門

設定網路

網路部分有做一篇 GCP 網路設定介紹，可以到這邊看 文章傳送門

設定 SSH 金鑰

(5) 設定完後右上拉開會顯示每個月多少金額

再次確認步驟的內容是否正確，確認無誤後點選建立，建立 VM

5. 檢查和建立後修改

建立好 VM 後 VM 執行個體可以看到建好的 VM，VM 建好是開啟的狀態，打勾選擇 VM 右邊三點或是右上角選擇要對 VM 做的動作 ( 刪除、停止、重設 )

點 VM 名子可以進入查看詳細資料

點編輯可以再次對 VM 做編輯，一樣可以做刪除、停止、重設等動作，但是 CPU、記憶體設定要停止 VM 才能更改，images 跟地區無法更改

GCP VM 建立完成

GCP-建立專案

發表於 2018-06-18 更新於 2020-04-01

本文章將介紹如何建立 GCP 專案

建立專案 ( GUI )

1. 進入 GCP，網址：https://cloud.google.com/

2. 登入 Google 帳號

3. 點擊畫面右上角控制台進入 ( 主頁有各種資訊可以查看 )

4. 第一次進入畫面，有專案點選選取，無專案點選建立

4.1. 選取專案 ( 承 4. 點選選取會出現以建立專案讓你選擇 )，1. 直接點擊專案名稱2下進入專案 2. 專案名稱前方3個點，點一下點選右下開啟進入專案

4.2. 建立專案 ( 承 4. 點選建立後會出現建立專案畫面 )，命名名稱和 ID 後點選建立完成

上圖點擊編輯後會出現 ID 的名稱可以更改，但是 ID 更改完專案建立後就無法再變更

5. 專案已建立，再次進入 GCP 會以最後離開專案為進入畫面，目前所在專案前方會打勾

( 在專案中選擇其他專案 ) 點左上 ▼ 會跑出選取專案，1. 直接點擊專案名稱2下進入專案 2. 專案名稱前方3個點，點一下點選右下開啟進入專案

( 在專案中新增其他專案 ) 右上角新增專案建立新專案

為你的專案命名，下方編輯可以更改專案 ID ( 同樣 ID 更改完專案建立後就無法再變更 )

6. 進入專案畫面，主頁專案資訊有一些預設訊息可以看右上自訂可以調整


7. 如果想改專案名稱或是刪除專案可以點右上三個點 → 專案設定

或選擇左上的三條槓 ( 導覽選單 ) → IAM 與管理員 → 設定，做專案更名或刪除

進入設定畫面，更改完名稱後點擊右邊儲存即可 ( ID 上述有說無法再更改 )

刪除專案，刪除專案需要 30 天刪除需要輸入專案 ID


建立專案 ( 指令 )

1. 在 GCP 控制台中，點擊右上角工具欄上的 Cloud Shell 圖標

2. 啟動 Cloud Shell ( 第一次啟動需點選 啟動 CLOUD SHELL 如下圖 )

PS：Cloud Shell 虛擬機裝載了所有您需要的開發工具。它提供了一個持久的5GB主目錄，並在 Google Cloud 上運行。只需使用瀏覽器或 Google Chromebook 即可完成中大部分（ 如果不是全部的話 ）工作。

Cloud Shell 啟動畫面

3. 開啟專案指令

4. 查看帳戶指令

可以看到使用者是誰，其實就是 google 登入帳號

5. 查看專案指令

顯示專案 ID ( ID 盡可能都是自己取比較好分類 )

6. 更換專案指令

建好後也可以從 GUI 左上點 ▼ 查看你的專案名稱跟 ID

專案建立完成
