control ( 控制台 )

taskmgr ( 工作管理員 )

secpol.msc ( 本機安全性原則 )

gpedit.msc ( 本機群組原則編輯器 )

certmgr.msc ( 憑證 )

explorer ( 檔案總管 )

notepad ( 開啟記事本 )

mspaint ( 小畫家 )
